# Movie Library (Личная кинотека)

**Автор:** Лейкер Данил Эдуардович

## Описание

Приложение для ведения личной коллекции фильмов с графическим интерфейсом (Tkinter).  
Позволяет:
- Добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга.
- Просматривать все фильмы в таблице.
- Фильтровать фильмы по жанру (точное совпадение) и году выпуска.
- Сохранять коллекцию в JSON-файл и загружать из него.
- Удалять выбранные фильмы.
- Проверять корректность ввода (год — число от 1900 до 2026, рейтинг — число от 0 до 10).

## Требования

- Python 3.6+
- Стандартные библиотеки: `tkinter`, `json`, `os`

## Запуск

```bash
python main.py
